# ChangeLog

## v1.0.0

- Initial release
- Supported audio formats: OGG, MP3, WAV (per-format toggles in settings)
- Auto-reads song metadata (Title, Artist, Album) from audio file tags
- Custom station icons (icon.svg) and built-in icon library
- Remember last played station across sessions
- Hot reload stations from settings panel
- Compatible with ExtendedRadio (auto-detection)
