# ChangeLog

## v1.0.0

- Initial release
- Drop `.ogg` files into `ModsData/SimpleRadio/YourStation/` to create custom radio stations
- Auto-reads song metadata from OGG Vorbis Comments (Title, Artist, Album)
- Settings panel showing loaded stations and songs count
- Quick button to open data folder from game settings
- Compatible with ExtendedRadio and MapExt
