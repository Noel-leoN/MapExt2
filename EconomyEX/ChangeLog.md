## v1.2.1 - UI and Logging Improvements

* **[Settings/UI]:** Added a "Refresh Status" button in the mod settings to manually update the system conflict detection status.
* **[Logging]:** Improved system initialization logging to clearly distinguish between standby mode and actual disabling.

---

### 主要改动

* **[设置与UI]：** 在模组设置中新增了“刷新状态”按钮，允许玩家手动更新系统冲突检测状态。
* **[日志]：** 优化了系统初始化日志，明确区分待机模式与实际禁用状态。
