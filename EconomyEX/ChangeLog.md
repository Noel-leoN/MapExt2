## v1.2.1 - Feature Sync and UI Improvements

* **[Pet Control]:** Added a pet management tool (NoDogs 2.0) that allows disabling pet spawning, preventing new pet generation, or removing all existing pets from the city.
* **[Traffic Optimization]:** Added "No Through-Traffic" option to disable background traffic spawning for better performance.
* **[Pathfinding/AI]:** 
    - Added adjustable pathfinding range for schools and leisure activities.
    - Added pathfinding optimization for personal cars and taxis.
    - Improved housing search system with configurable throughput limits.
* **[Economy]:** Optimized shopping behavior and household resource demand calculation.
* **[Settings/UI]:** Added a "Refresh Status" button in the mod settings to manually update the system conflict detection status.

---

### 主要改动

* **[宠物管理]：** 新增宠物管理工具 (NoDogs 2.0)，可禁止宠物外出、阻止新宠物生成，或一键清除城市中的所有宠物。
* **[交通优化]：** 新增“禁止过境交通”选项，可禁用背景生成的过境流量以提升性能。
* **[寻路与 AI]：** 
    - 新增学校招生和休闲活动的寻路范围调整。
    - 新增私家车和出租车的寻路优化。
    - 优化了住房搜索系统，增加了可配置的搜索吞吐量上限。
* **[经济系统]：** 优化了购物行为逻辑与家庭资源需求计算。
* **[设置与UI]：** 在模组设置中新增了“刷新状态”按钮，允许玩家手动更新系统冲突检测状态。
