# SimpleBrush

A lightweight Cities: Skylines 2 mod that unlocks hidden natural resource brushes and lets you restore depleted resources with one click.

## Features

**Resource Brush Unlock**
- Reveals hidden Ore, Oil, Fertile Land, and Ground Water brushes in the Terraforming toolbar
- Works in both Game and Editor modes
- Zero Harmony patches required - uses pure native PrefabSystem API

**Restore Depleted Resources**
- One-click buttons in Mod Settings to reset used resources back to full
- Individual buttons for Fertility, Ore, Oil, and Fish
- "Restore All" button to reset everything at once
- Confirmation dialogs to prevent accidental clicks

## Compatibility

- **MapExt2**: Fully compatible. When MapExt2 is active, resource operations are transparently redirected to the extended CellMap buffer
- **ExtraLandscapingTools**: Functional overlap but no conflicts. Recommend using one or the other
- **No dependencies**: Zero external library requirements (no ExtraLib, no Harmony)

## Installation

Subscribe on [Paradox Mods](https://mods.paradoxplaza.com/) or place the DLL in your local Mods folder.
